# Simple Tic-Tac-Toe Game

Simple tic-tac-toe with AI made out of C++ and SFML(2.0) Library

## Some Game Info:
	- Game Still in development
	- PVP(Just Press Num1 in the init screen) mode
	- PVC mode

## CONTROLS: 
	Q | W | E
	--+---+--
	A | S | D
	--+---+--
	Z | X | C

	Num2 to Restart the game;

## BUILD:
	- Requires C++14 
	- Requires SFML library
	- make and enjoy
